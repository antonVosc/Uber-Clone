# Uber-Clone
This is an Uber clone app which I have made using Vue 3, Tailwind CSS, Node JS, Google Maps, PWA and JavaScript.


To run the project, run:

1. 
```
git clone https://github.com/antonVosc/Uber-Clone.git
```

2. 
```
cd Uber-Clone/
```

3. 
```
npm install
```

4. 
```
cd server/
```

5. 
```
npm install
```

6. 
```
cd ../
```

7. 
```
npm run serve
```
